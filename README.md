# Pandas Tutorial

## Iniciar entorno
utilizando virtualenv
~~~
virtualenv -p /usr/bin/python3.6 env
source env/bin/activate
pip install -r requirements.txt
~~~
Utilizando Conda
~~~
conda create -n pandas-tutorial python=3.6
source activate pandas-tutorial
pip install -r requirements.txt
~~~

## Ejecutar jupyter
~~~
source env/bin/activate
jupyter notebook
~~~
